Un navegador Web es un software de aplicación que sirve para abrir páginas Web tanto en una ruta local (como el disco rígido) o provenientes de la Internet. Se les conoce como “navegadores” o “exploradores”, a partir de la metáfora de que la Red es un lugar vasto y abarrotado, para el que se necesita de una plataforma. En ese sentido, los navegadores Web nos permiten “entrar” a Internet y visualizar distintos contenidos a partir del ingreso de direcciones URL o del empleo de servicios online de búsqueda de datos

Una de las prácticas de excelencia en la administración de empresas es hacer que la tecnología trabaje para el negocio. Esto quiere decir que no debemos limitarnos a que la tecnología sea vista simplemente como una herramienta para la administración de las operaciones, sino que además funcione como un instrumento que nos distinga de nuestra competencia. Tal es el caso del posicionamiento en buscadores, una tecnología que nos puede ayudar a impulsar nuestro negocio, al promover activamente nuestros productos y servicios a través de una página web.
